# 🌍 Real-Time Air Quality Analyzer

## 📌 Overview
The **Real-Time Air Quality Analyzer** provides live AQI data for any city using **OpenWeatherMap API** and offers **AI-generated health recommendations** via **Gemini AI**. This tool helps users understand air quality levels and take necessary precautions to stay safe.

## 🚀 Features
- 🌡️ **Live AQI Data**: Fetches real-time air quality index (AQI) using **OpenWeatherMap API**.
- 🏥 **Health Measures & Precautions**: AI-generated personalized health recommendations based on AQI level.
- 🎨 **User-Friendly Interface**: Clean, responsive, and interactive design.
- 🌎 **Global Coverage**: Works for cities worldwide.
- 🔍 **Search Functionality**: Enter a city name to fetch instant AQI details.
- 📊 **Categorized AQI Levels**: Color-coded AQI display (Good, Moderate, Unhealthy, etc.)

## 🛠️ Technologies Used
- **HTML, CSS, JavaScript** - Core web development stack.
- **OpenWeatherMap API** - Fetches AQI & geolocation data.
- **Gemini AI API** - Provides health recommendations based on AQI.

## 🏥 Health Recommendations (AI-Powered)
- **Health Measures** 🏥: AI suggests protective steps based on AQI severity.
- **Precautions** ⚠️: Tips to reduce exposure to pollution.
- **Recommendations** ✅: Actions to improve air quality and personal safety.
